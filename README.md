# 🤖 Enterprise Integration Sales Engineer Agent

An AI-powered agent that automates sales engineering workflows using Gemini AI, ChromaDB, and RAG.

## 🚀 Features
- Extracts client pain points, tech stack & outcomes from meeting transcripts
- RAG system using ChromaDB to retrieve relevant case studies
- Suggests technical architectures based on client needs
- Auto-generates Solution Design Documents (.docx)

## 🛠️ Tech Stack
- Python
- Gemini AI
- ChromaDB
- RAG (Retrieval Augmented Generation)
- Streamlit
- Python-docx

## ⚙️ Setup & Installation
1. Clone the repo
   git clone https://github.com/Shree-dh/enterprise-integration-sales-agent.git
2. Install dependencies
   pip install -r requirements.txt
3. Add your API key in .env file
   GEMINI_API_KEY=your_key_here
4. Run the app
   streamlit run app.py
